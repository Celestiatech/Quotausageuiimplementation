
  # Free Auto Apply CV - AI-Powered Job Application Automation

  Free auto apply CV tool that automates job applications on LinkedIn and Indeed. Save hours with AI-powered resume matching, smart application tracking, and automated job search.

  ## 🚀 Features

  - **100% Free Auto Apply** - Automatically apply to hundreds of jobs
  - **LinkedIn & Indeed Integration** - Auto-apply across multiple platforms
  - **AI Resume Matching** - Smart matching with job requirements
  - **Application Tracking** - Track all your applications in one dashboard
  - **Smart Filtering** - Apply only to relevant jobs
  - **Resume Management** - Multiple resume versions for different roles

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
